# BHUINFO
A News/Magazine web application for Bingham University.
Language:ASP.NET 4.6 C#
Framework:Entity Framework

This Application is divided into 6 seperate projects
1.Datacontext project = This holds the database context for each object.
2.Objects project = This consist of entities that make up an object.
3.Service Project = This consist of 
a.MD5 Encrytion class
b.Email Sender
c.Strip HTML Handler
d.Date time helper
e.Enum
f.Config
g.File Uploader
4.Web project= This consist of the web views and theri respective master pages and their controllers
5.Factory Project = This consist of methods that aid the retrival and calculation of data.
6.Mapping Project = This maps each object entity to the colums and specifies the relationship between colums and tables
